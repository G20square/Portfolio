# Personal Portfolio Website

This is a modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features
- **Modern Dark Theme**: Professional and clean design.
- **Responsive Layout**: Works seamlessly on desktop and mobile devices.
- **Interactive Elements**: Smooth scrolling, skill bar animations, and project hover effects.
- **Clean Structure**: Easy to customize and extend.

## Project Structure
```
portfolio/
├── index.html      # Main HTML file
├── style.css       # Main stylesheet
├── script.js       # JavaScript for interactivity
└── assets/         # Directory for images
```

## How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```
2. Open `index.html` in your web browser.
   - You can simply double-click the file or use a live server extension in VS Code.

## Deployment

### GitHub Pages
1. Push your code to a GitHub repository.
2. Go to the repository **Settings**.
3. Navigate to the **Pages** section.
4. Under **Source**, select `main` (or `master`) branch.
5. Click **Save**. Your site will be live at `https://yourusername.github.io/portfolio/`.

### Netlify
1. Log in to Netlify and drag-and-drop your project folder.
2. Alternatively, connect your GitHub repository for continuous deployment.

## customization
- **Content**: Edit `index.html` to update your information, projects, and skills.
- **Styles**: Modify `style.css` to change colors, fonts, or layout.
- **Images**: Add your images to the `assets/` folder and update the `src` attributes in `index.html`.
